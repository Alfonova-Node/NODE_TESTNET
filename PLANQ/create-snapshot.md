## How To Create or Provide Public Snapshot

```bash
sudo apt install lz4
cd $HOME/.planqd
sudo systemctl stop planqd
tar -cf - data | lz4 > /var/www/snapshot/planq/planq-snapshot-$(date +%Y%m%d).tar.lz4
sudo systemctl start planqd
```
